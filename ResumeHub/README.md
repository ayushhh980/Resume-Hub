# ResumeHub - Resume Builder

Mobile-friendly Resume Builder web app.

## Tech Stack
- Frontend: HTML/CSS/JS (responsive)
- Backend: Node.js/Express
- DB: MongoDB Atlas (free)

## Setup
1. MongoDB Atlas: Create free cluster, get MONGO_URI.
2. Backend: `cd backend && npm install && cp ../.env.example .env && npm start`
3. Frontend: Open `frontend/index.html` (connects to backend)
4. Templates: IT, Marketing, Finance, Education, Healthcare, Creative, General, Internship.

## Features
- User auth/login
- Multi-section input forms
- Drag-drop editor
- Real-time preview
- PDF export
- Responsive/dark mode
