# ResumeHub - Resume Builder

Mobile-friendly Resume Builder web app.

## Tech Stack
- Frontend: HTML/CSS/JS (responsive)
- Storage: Browser localStorage/sessionStorage
- Optional Cloud Save: GitHub Repository file (GitHub REST API)

## Setup
1. Entry point is root [index.html](index.html).
2. Open the deployed GitHub Pages URL (recommended) or open [index.html](index.html) locally.
3. Login/Register works in static mode (local browser profile only).
4. Fill resume in dashboard and use:
	- Save Locally (browser storage)
	- Save to GitHub (writes JSON to your repo via token)
	- Load from GitHub (reads the same JSON file)

## Features
- Static auth/login (no backend)
- Multi-section input forms
- Drag-drop editor
- Real-time preview
- PDF export
- Responsive/dark mode
- GitHub-based resume JSON save/load (optional)
